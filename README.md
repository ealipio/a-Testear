## test using karma, Jasmine over angularjs application
I just playing with angularjs 1.6.7 and test functionality using jasmin and then run all test suite using karma
### Setup

`npm i`

`node server.js` or `nodemon server.js`

### Test

`npm i -g karma-cli`

`karma start`
